# Spring Data MyBatis 
                                                                                                                                                              
REAP 默认持久化层，fork 自 [spring-data-mybatis](https://github.com/hatunet/spring-data-mybatis)。

## 区别

* 支持 Spring Boot 2.x
* 内置提供可扩展的 ID 生成机制
* 使用 JPA 标准标注
* 不支持 Entity association（@OneToOne、@OneToMany、@ManyToMany）
* 不支持 Auditing
